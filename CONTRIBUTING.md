# Contributing

- **Branches**: Work on the `dev` branch. The `main` branch is stable and only updated via pull requests from `dev`. See [README → Development and release](README.md#development-and-release) for the branch model and how to protect `main`.
- **Changelog**: Update the `[Unreleased]` section in [CHANGELOG.md](CHANGELOG.md) with your changes.
- **Releasing**: Use the **Release** GitHub Action (Actions → Release → Run workflow) and then merge the created PR. See [README → Releasing a new version](README.md#releasing-a-new-version).
